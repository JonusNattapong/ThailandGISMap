# Contributing to Thailand Universal GIS Data Stack

ขอบคุณที่สนใจช่วยเหลือและพัฒนาโปรเจกต์นี้ เพื่อให้การพัฒนาเป็นไปอย่างราบรื่น โปรดปฏิบัติตามแนวทางดังนี้:

## ขั้นตอนการส่งผลงาน

1. Fork โปรเจกต์นี้
2. สร้าง Branch สำหรับฟีเจอร์หรือการแก้ไขของคุณ (`git checkout -b feature/AmazingFeature`)
3. Commit การเปลี่ยนแปลงของคุณ (`git commit -m 'Add some AmazingFeature'`)
4. Push ไปยัง Branch ของคุณ (`git push origin feature/AmazingFeature`)
5. เปิด Pull Request

## รูปแบบการปรับปรุงข้อมูล

* หากมีการอัปเดตไฟล์ GeoJSON โปรดระบุแหล่งที่มาและวันที่เข้าถึงข้อมูลให้ชัดเจน
* โปรดตรวจสอบความถูกต้องของพิกัดและลำดับชั้นการปกครองก่อนส่งการเปลี่ยนแปลง

ขอขอบคุณทุกการสนับสนุนครับ
